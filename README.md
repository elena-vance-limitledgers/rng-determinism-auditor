# RNG Determinism & RTP Variance Stress-Tester

Internal tool used by the LimitLedgers Audit Team to verify empirical RTP against theoretical B2B provider claims. 

Designed to detect **Max Bet Truncation** anomalies typically found in non-licensed (Spoofed) API environments. By simulating millions of cryptographic iterations, we isolate and flag unjustified variance drops during high-liquidity (VIP) sessions.
